# 用户案例墙

本文档用于收录 Sourcelin Blog 的真实部署案例、二次开发案例和课程项目案例。

目的有两点：

1. 让新用户快速判断项目是否真的被使用
2. 为后续 README 展示、社区传播和案例征集提供统一入口

## 如何提交案例

欢迎通过 Issue 或 PR 提交你的案例信息，建议包含：

1. 站点名称
2. 访问地址
3. 使用场景
4. 部署方式
5. 是否基于原版或二次开发
6. 可选截图

提交入口：

- Issue：[https://gitee.com/my_lyq/sourcelin-cloud-blog/issues](https://gitee.com/my_lyq/sourcelin-cloud-blog/issues)
- Pull Request：[https://gitee.com/my_lyq/sourcelin-cloud-blog/pulls](https://gitee.com/my_lyq/sourcelin-cloud-blog/pulls)

## 案例模板

```text
站点名称：

```
