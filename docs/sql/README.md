# SQL 初始化说明

本目录保留当前项目长期维护的数据库初始化脚本。

## 文件说明

- `sourcelin-cloud.sql`：业务库初始化脚本，包含当前实际表结构和最小必要基础数据。
- `sourcelin-config.sql`：Nacos 配置库初始化脚本。
- `sourcelin-seata.sql`：Seata 事务库初始化脚本。

## `sourcelin-cloud.sql` 数据边界

保留的基础数据：

- 系统配置、字典、菜单、角色、部门、岗位、管理员账号和关联关系。
- 定时任务基础定义。
- 博客前台站点基础配置和导航栏。
- 内容举报表结构、后台“内容举报”菜单和 `blog:report:list/query/edit/remove` 权限基础数据。

不写入初始化脚本的数据：

- 文章、分类、标签、评论、说说、树洞、友链等业务内容。
- 点赞、收藏、关注、已读记录、评论机审记录等运行态数据。
- 文件中心记录、访问日志、操作日志、代码生成元数据。

## 维护规则

- 更新表结构时，以当前数据库实际结构为准重新生成。
- 不提交真实环境账号、密码、token、IP、文件访问地址等敏感或环境绑定数据。
- 初始化脚本必须能导入到空库并创建完整表结构。

- sourcelin-config.sql 已移除 Nacos 历史配置数据，仅保留当前有效配置；示例账号、数据库、对象存储口令均使用占位值，导入后按本地环境修改。
