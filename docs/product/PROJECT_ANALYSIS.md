## 一、项目概述

Sourcelin Blog（圆圈博客）是一套面向个人写作与生活记录的博客平台，采用前后端分离架构，支持多端访问与内容管理。项目基于sourcelin Sourcelin-Cloud v3.6.3 开发。

### 1.1 项目定位
- ✍️ **个人博客平台** - 记录生活、写作随笔与分享见闻
- 📚 **内容创作空间** - 文章、分类、标签、评论等完整功能
- 🧩 **小团队内容中心** - 支持多角色内容协作与管理
- 🗂️ **长期内容归档** - 让内容沉淀与可持续维护

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Spring Boot 2.7.18 + Spring Cloud 2021.0.9 + Spring Cloud Alibaba 2021.0.6.1 |
| 前端 | 博客前台：Vue 3 + TypeScript + Naive UI；管理后台：Vue 3 + TypeScript + Element Plus |
| 数据库 | MySQL 8.0 + Redis |
| 微服务组件 | Nacos + Sentinel + Seata |
| 文件存储 | MinIO |

## 四、开发建议

### 4.1 架构优化
1. **前端分离**: 博客前台(sourcelin-ui-platform)和管理后台(sourcelin-ui-admin)已分开
2. **移动端**: 建议使用 uni-app 开发小程序和 H5
3. **性能优化**: 可考虑引入 Redis 缓存、CDN 加速

### 4.2 安全建议
1. 生产环境需配置 HTTPS
2. Sa-Token 与 Nacos 配置需要统一收敛，避免回退到旧 JWT 方案
3. 接口需要添加限流机制
4. 敏感数据需要加密存储

### 4.3 部署建议
1. 使用仓库提供的 `docker-compose.example.yml` 和 `.env.example` 作为公开部署示例
2. 生产环境建议使用 K8s
3. 配置日志聚合系统
4. 启用监控告警
