# 对外传播内容

`docs/promotion/` 用于存放面向外部社区传播的内容文档。

这类文档和常规实现文档不同，重点不是描述代码结构或部署步骤，而是服务于：

- 开源中国、掘金、知乎、CSDN、博客园等图文平台发文
- Gitee 动态、GitHub Release、社区帖子的长文素材沉淀
- 对外介绍项目定位、差异化能力、AI 协作实践和版本更新亮点

## 放什么

- 可直接改写后发布到社区的文章草稿
- 项目对外介绍稿
- 版本发布说明扩展稿
- AI Coding 实践总结

## 不放什么

- 内部设计过程文档
- 生产环境部署细节
- 仅供仓库维护者使用的临时操作记录

这类内容分别维护在：

- `docs/architecture/`
- `docs/deployment/`
- `docs/internal/`
- `docs/guides/`
